# Back to the Moon

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクターおじいさんを矢印キーで左右に動かし，スペースキー押下で言弾(ことだま)を飛ばし、月に帰ろうとするかぐや姫を引き留めるゲームです。
* 参考URL: [竹取物語 -wikipedia](https://ja.wikipedia.org/wiki/%E7%AB%B9%E5%8F%96%E7%89%A9%E8%AA%9E)

## ゲームの遊び方
* 矢印キーで左右移動、スペースキー押下で言弾を飛ばす。
* かぐや姫は徐々に月に近づいていくので、言弾を当てて引き留めよう。
* ゲーム中に落ちてくる隕石がかぐや姫に当たるとゲームオーバーになります。おじいさんが当たるとスタン状態になり、しばらく動けなくなります。
* かぐや姫に追いつくことができたらゲームクリア
* かぐや姫が月に到達してしまったらゲームオーバー

## ゲームの実装
### 共通基本機能
* キャラクターの描画、移動操作、言弾の発射、ゲームクリア、ゲームオーバーの判定(生成AIを使用し作成)

### 分担追加機能
* キャラクターチェンジ（担当：なかむら）：おじいさんやおばあさんなどのキャラクターの切り替え、キャラクターごとに性能が変化。パラメータ：足の速さ(左右移動の速さ。あとから追加の可能性あり)。Sキーで切り替え
* 隕石が落ちてくるロジックの追加（担当：ろっく）:落ちてきた隕石はかぐや姫に当たったら即ゲームオーバー、おじいさんに当たったらスタン状態になる
* 弾を言弾(吹き出し)に変更と吹き出しの音声・BGMの追加（担当：せつ）：スペースを押すと吹き出しを発射する。その際にセリフを再生する。また、BGMを追加する。
* 追いつくまで複数回当てる(弾を充てるとかぐや姫が少し近づいてくる)+ゲームオーバーまでのゲージ追加 (担当：あいば): 現状一度当てたらゲームオーバーなところを複数回当てたらゲームクリアにする。当てるごとにかぐや姫が近づき、当てた回数カウンタが1増加する。数秒間当てられないままだと、かぐや姫が遠ざかり減少する。また、ゲームオーバーまでの進行度ゲージを追加する。
* ゲーム開始時であらすじの語りを入れる・キャラクターなど画像の追加（担当：はらだ）

### メモ
* ファイル名は`main.py`にしている。
* 言弾は発射する弾のことです。
